The astronomer's assistant calculates the user's location, available for observing the planets, taking into account the weather, the possible and best observation time.
Program uses geolite2PP library, source: https://www.ccoderun.ca/GeoLite2++/api/index.html
Program uses GgeoLite2-City.mmdb database, source: https://dev.maxmind.com/geoip/geolite2-free-geolocation-data?lang=en#databases
